## **Introducción**  
Este proyecto es un portafolio profesional desarrollado en **Astro**. Su objetivo es destacar las habilidades, experiencia y proyectos de **David Cuevas Gil**, un Técnico Superior en Desarrollo Web.  

El portafolio está diseñado para ser una herramienta funcional y estética que muestre su trayectoria profesional, conocimientos técnicos y pasión por el desarrollo web. La motivación detrás de este proyecto radica en la necesidad de consolidar en un solo lugar las experiencias y capacidades, facilitando su presentación a futuros empleadores o colaboradores.  

### **Objetivos del proyecto**  
- Crear una plataforma interactiva para presentar la información profesional.  
- Incorporar buenas prácticas en diseño y desarrollo web.  
- Usar **Astro** para optimizar la carga y organización del sitio.  

---

## **Funcionalidades y Tecnologías**  

### **Componentes actuales**  
1. **Header**: Una cabecera con el nombre y descripción profesional del autor.  
2. **Menú lateral**: Sistema de navegación fijo con enlaces a las secciones clave del sitio.  
3. **Sección "Sobre mí"**: Información detallada sobre la experiencia, motivaciones y conocimientos lingüísticos.  
4. **Sección "Habilidades"**: Visualización de habilidades técnicas con un sistema de puntuación.  
5. **Sección "Experiencia"**: Descripción estructurada de roles laborales más relevantes.  
6. **Sección "Proyectos"**: Galería interactiva de proyectos con descripciones y tecnologías utilizadas.  
7. **Sección "Formación"**: Registro de cursos y estudios académicos, con enlaces a certificados.  
8. **Sección "Referencias"**: Listado de referencias profesionales destacadas.  
9. **Sección "Contacto"**: Información de contacto, formulario y enlaces a redes sociales.  

### **Tecnologías utilizadas**  
- **Astro**  
- **Tailwind CSS**  
- **HTML**  
- **JavaScript**  
- **Figma**

## **Guía de Instalación**  
1. Clonar el repositorio.  
2. Instalar dependencias con `npm install`.  
3. Ejecutar el proyecto localmente con `npm run dev`.  

---

## **Guía de Uso**  
Navegar por las diferentes secciones utilizando el menú lateral para explorar habilidades, proyectos, experiencia, formación y más.
